# تطبيق إشارات الخيارات الثنائية للهاتف المحمول

تطبيق Flutter للهاتف المحمول لإشارات الخيارات الثنائية (عربي أولاً، واجهة داكنة).

## الإعداد

1. **تثبيت مكتبات Flutter:**
   ```bash
   flutter pub get
   ```

2. **تكوين Firebase:**
   - أضف `google-services.json` (Android) إلى `android/app/`
   - أضف `GoogleService-Info.plist` (iOS) إلى `ios/Runner/`
   - أو قم بتكوين Firebase يدوياً

3. **تكوين عنوان API الأساسي:**
   - التطبيق يستخدم `http://localhost:3000` افتراضياً
   - لمحاكي Android، استخدم `http://10.0.2.2:3000`
   - لمحاكي iOS، استخدم `http://localhost:3000`
   - للجهاز الفعلي، استخدم IP جهاز الكمبيوتر

4. **تشغيل التطبيق:**
   ```bash
   flutter run
   ```

## الميزات

- ✅ دعم العربية RTL (اللغة الافتراضية)
- ✅ تصميم داكن جريء مع لهجات نيون
- ✅ اختيار المنصة (Quotex، Pocket Option)
- ✅ تدفق التحقق من الحساب
- ✅ خلاصة الإشارات (مقفلة/مفتوحة حسب الحالة)
- ✅ حاسبة المخاطر
- ✅ الإشعارات الفورية (FCM)
- ✅ الإعدادات (اللغة، تغيير المنصة، تسجيل الخروج)

## هيكل المشروع

```
lib/
├── core/
│   ├── config/          # تكوين التطبيق
│   ├── localization/    # الترجمات العربية/الإنجليزية
│   ├── providers/       # موفرو Riverpod
│   ├── routing/         # تكوين GoRouter
│   ├── services/        # خدمات API
│   └── theme/           # التصميم الداكن
├── features/
│   ├── home/           # الشاشة الرئيسية مع الإشارات
│   ├── onboarding/     # شاشات اختيار المنصة والتحقق
│   ├── risk/           # حاسبة المخاطر
│   ├── settings/       # شاشة الإعدادات
│   └── signals/        # تفاصيل الإشارة
└── main.dart           # نقطة دخول التطبيق
```

## الاختبار

استخدم معرفات الحسابات التجريبية من البيانات المملوءة:
- `TEST_ACTIVE_123` (QUOTEX) - VERIFIED_ACTIVE
- `TEST_NO_DEPOSIT_456` (QUOTEX) - VERIFIED_NO_DEPOSIT

## التدفق

1. **Splash** → التحقق من الرمز → التوجيه
2. **اختيار المنصة** → اختر Quotex أو Pocket Option
3. **التحقق من الحساب** → أدخل معرف الحساب
4. **نتيجة التحقق** → عرض الحالة + CTA
5. **الرئيسية** → خلاصة الإشارات (إذا كان verified active)
6. **تفاصيل الإشارة** → عرض الإشارة + فتح الوسيط
7. **حاسبة المخاطر** → حساب حجم المركز
8. **الإعدادات** → تبديل اللغة، تغيير المنصة، تسجيل الخروج
